# 🛡️ Insurance Fraud Detection Dashboard

An end-to-end **Data Analytics Project** focused on identifying suspicious insurance claims using **Python, SQL, and Power BI**. This project analyzes claim records, uncovers fraud patterns, and presents business insights through an interactive dashboard.

![Insurance Fraud Dashboard](Insaurance%20Fraud%20Dashboard.png)
---

## 📌 Project Overview

Insurance fraud causes major financial losses for companies every year. The goal of this project is to:

- Detect high-risk claims  
- Analyze fraud trends by state, incident type, and demographics  
- Measure fraud loss amount  
- Help investigators prioritize suspicious claims

- Build an interactive dashboard for decision-making  

---

## 🛠️ Tech Stack

- **Python** – Data Cleaning, EDA  
- **Pandas / NumPy** – Data Processing  
- **Matplotlib / Seaborn** – Visualization  
- **SQL (MySQL)** – Business Queries & KPIs  
- **Power BI** – Interactive Dashboard  
- **Excel / CSV Dataset**

## 🔍 Data Analysis Workflow

### 1️⃣ Data Cleaning & Preprocessing

Performed using Python:

- Standardized column names  
- Handled missing values  
- Filled categorical null values with mode / "Unknown"  
- Filled numeric null values using median  
- Removed duplicates  
- Created new features:  
  - `fraud_flag`  
  - `age_group`  

---

### 2️⃣ Exploratory Data Analysis (EDA)

Key analysis performed:

- Fraud vs Legitimate claim distribution  
- Claim amount comparison  
- Fraud by incident type  
- Fraud by state  
- Fraud by age group  
- Monthly claim trends  
- Authorities contacted during claims  
- High claim amount fraud cases  

---

### 3️⃣ SQL Business Queries

Used SQL to generate insights such as:

- Total claims  
- Total fraud claims  
- Fraud rate %  
- Fraud by incident type  
- Fraud by state  
- Average claim amount (fraud vs genuine)  
- Top occupations involved in fraud  
- Top costly fraud claims  
- Total fraud loss amount
